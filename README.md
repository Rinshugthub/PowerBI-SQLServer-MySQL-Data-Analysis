# PowerBI-SQLServer-MySQL-Data-Analysis

## Project Overview
This is an end-to-end Business Intelligence project developed using Power BI, SQL Server, and MySQL. The project demonstrates data extraction, transformation, SQL querying, DAX calculations, and interactive dashboard development to generate meaningful business insights.

## Tools & Technologies
- Power BI Desktop
- SQL Server
- MySQL Workbench
- SQL
- DAX
- Power Query

## Project Workflow
1. Imported raw data into SQL Server and MySQL.
2. Performed data cleaning and transformation.
3. Used SQL queries and LEFT JOIN to prepare data.
4. Imported processed data into Power BI.
5. Created DAX measures and KPIs.
6. Designed interactive dashboards.
7. Published the final report.

## Dashboard Features
- Interactive KPI Cards
- Dynamic Filters & Slicers
- Business Performance Analysis
- Trend Analysis
- Data Visualization
